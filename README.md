# information-theory-tool
This toolbox contains Matlab source code for basic computations in information theory.

You can compute: 

-   entropy 
-   conditional entropy 
-   mutual information 
-   channel capacity 
-   rate distortion

Functions and several examples are provided, as well as their experiment results in figures.
